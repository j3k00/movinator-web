## Description

This program is written in Java and it changes the following instructions into
a sequence of mov instructions

## Functionallity

Instructions that has been implemented:

* Add 32 bit (addl)
* Sub 32 bit (subl)
* Inc 32 bit (incl)
* Dec 32 bit (decl)
* Push 32 bit (push)
* Pop 32 bit (pop)
